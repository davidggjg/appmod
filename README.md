# AppStore

חנות אפליקציות אישית — בנויה עם React + Vite.

## העלאה לגיטהאב Pages

1. צור ריפו חדש בגיטהאב (שם לפי בחירתך)
2. העלה את תוכן הזיפ לריפו
3. לך ל־Settings → Pages → Source: **GitHub Actions**
4. כל push לענף main יבנה ויפרסם את האתר אוטומטית

## כניסה לפאנל ניהול

הקלד את הקוד הסודי ישירות בסרגל החיפוש.
